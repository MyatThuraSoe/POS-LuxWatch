================================================================================
LUXWATCH - FRONTEND DEVELOPMENT ROADMAP & REQUIREMENTS
Watch Shop POS and Management System
================================================================================
PROJECT OVERVIEW
--------------------------------------------------------------------------------
LuxWatch is a comprehensive Watch Shop POS and Management System for:
- Classic Watches
- Smart Watches
User Roles:
- ADMIN: Full system access, configuration, all reports
- OWNER: Business operations, reports, staff management
- EMPLOYEE (Cashier/Seller): POS sales, customer service, basic inventory
================================================================================
TECHNOLOGY STACK
================================================================================
CORE FRAMEWORK:
- React 19+ (with TypeScript)
- Vite (Build tool & Dev server)
UI & STYLING:
- Tailwind CSS (Utility-first CSS framework)
- Shadcn UI (Component library built on Radix UI)
- Lucide React (Icon library)
- Custom SCSS modules (for complex animations)
STATE MANAGEMENT:
- Zustand (Global application state)
- TanStack Query v5 (Server state, caching, synchronization)
FORMS & VALIDATION:
- React Hook Form (Form handling)
- Zod (Schema validation)
- @hookform/resolvers (Integration)
ROUTING:
- React Router v6+ (Navigation)
- Protected routes with role-based access
DATA VISUALIZATION:
- Recharts (Charts & graphs)
- TanStack Table (Advanced data tables)
HTTP CLIENT:
- Axios (API requests with interceptors)
UTILITIES:
- date-fns (Date manipulation)
- numfmt (Number formatting)
- barcode-vue / quagga (Barcode scanning)
- react-to-print (Receipt printing)
NOTIFICATIONS:
- Sonner (Toast notifications)
- Custom confirmation dialogs
================================================================================
DESIGN PHILOSOPHY
================================================================================
Visual Style:
- Modern, minimal, professional
- Clean dashboards with clear hierarchy
- Consistent spacing (8px grid system)
- Consistent typography (Inter font family)
- Subtle animations (200-300ms transitions)
- High contrast for accessibility
Responsive Breakpoints:
- Desktop: 1280px+ (Primary target - POS stations)
- Laptop: 1024px - 1279px
- Tablet: 768px - 1023px (Secondary support)
- Mobile: < 768px (Admin views only, not POS)
Color Palette:
- Primary: Deep Blue (#1e40af)
- Secondary: Slate Gray (#64748b)
- Success: Emerald (#10b981)
- Warning: Amber (#f59e0b)
- Danger: Rose (#f43f5e)
- Background: White / Light Gray (#f8fafc)
- Surface: White with subtle shadows
================================================================================
